Tương Tác
    - Gọi Nhân vật : bạn hãy click vào đây tìm các File có biểu tượng  .
        VD: Tôi có 1 file Merlin.acs
                Tôi chỉ cần nhớ tên "Merlin"
                Sau đó khởi động AI.exe > Gõ vào     
                ::agent merlin;;          => "Nó" sẽ trả lời : "OK !"  
                        ( Có thể máy bạn chỉ có 1 File merlin.acs nếu bạn muốn tải về thêm thì hãy Click vày đây )
                 ==> Xong .
                Còn nếu bạn muốn tắt nó đi thì gõ vào :
                                    ::agent null;;         => "Nó" sẽ trả lời : "OK !"  
        VD2: Bạn có 1 File agent tên James.acs nằm trong ổ D
                Hãy gõ vào :
                                    ::agent d:\james.acs;;             
      
    - Chức năng phát âm Tiếng Việt :
            Để bật tính năng này gõ :
                    ::sound on;;                => "OK !"
            Để tắt đi thì gõ :
                    ::sound off;;                => "OK !"
