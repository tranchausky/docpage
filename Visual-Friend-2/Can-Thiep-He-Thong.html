Chức Năng Quản Lí Hệ Thống

        Lưu ý :
                    Tới đây tôi xem như các bạn đã đọc HDSD 1 và 2 
                    nên sẽ sử dụng WildCard * để hướng dẫn 
 

    1. Ẩn các ổ đĩa :

        1.1 Ẩn từng ổ xác định :
                CT :         *ẩn ổ x*            [ x là tên ổ đĩa cần ẩn ]
                        VD:    "tao muốn ẩn ổ c" => ẩn ổ C dùm bạn
                                  "hãy ẩn ổ d,e" => Nó chỉ ẩn 1 ổ thôi, ở đây nó sẽ chỉ nhận ẩn ổ D
        1.2 Ẩn tất cả các ổ đĩa trong máy :
                CT :        *ẩn tất cả các ổ*
                        VD:    "làm ơn ẩn tất cả các ổ đĩa trong máy"
        1.3 Hiện tất cả các ổ đĩa trong máy :
                CT :        *hiện tất cả các ổ*
                        VD:    "hãy hiện tất cả các ổ đĩa trong máy"


    2. Cấm truy nhập ổ đĩa :

        2.1 Cấm trên từng ổ xác định :
                CT:        *không*truy nhập ổ x*        [ x là tên ổ đĩa muốn cấm ]
                              *cấm*truy nhập ổ x*           [ Lưu ý là "truy nhập" chứ không phải "truy cập" ]
                        VD :    "không truy nhập ổ C" 
                                   "cấm người sử dụng truy nhập ổ d"
        2.2 Cấm tất cả các ổ :
                CT:        *cấm truy nhập tất cả các ổ*
        2.3 Cho phép truy nhập tất cả :
                CT:        *cho phép truy nhập tất cả các ổ*


    3. Task Manager :

        3.1 Cấm task manager ( nhấn Ctrl-Alt-Delete sẽ hiện lên 1 cái thông báo tổ bố cấm vào ^_^ )

        3.2  Cho phép :

    10. Process Manager ( Xem các ứng dụng đang chạy trên hệ thống của bạn và chấm dứt (nếu cần) )

        10.1 Xem: 
                CT:         *process list*
                               *danh sách ứng dụng*
        10.2 Hủy ứng dụng ( Kill Process) :
                Bạn phải gõ vào từ process list hoặc danh sách ứng dụng để có được ID của ứng dụng cần hủy rồi gõ vào
                Kill “số ID của ứng dụng” ( không có dấu ngoặc kép,không cần viết hoa )
                Hủy “số ID của ứng dụng” ( không có dấu ngoặc kép,không cần viết hoa )
                VD : trong process list tôi thấy có iexplore có ID là 236
                kill 236             [ => Đơn giản quá ! ]
